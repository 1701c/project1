# Whats on Where?
Web app that helps a viewer find movies and TV shows across multiple streaming services.

Try app at https://1701c.github.io/project1/

<h1>Built with</h1>
<ul>
  <li>Bulma- CSS FrameWork</li>
  <li>Firebase- Database and User Auth</li>
  <li>Vanilla JS</li>
</ul>
